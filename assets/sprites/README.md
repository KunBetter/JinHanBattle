## ⚠️ 注意事项

- 当前项目 **汉金之战 (JinHanBattle)** 使用**程序化生成**（`assets.js` 中 Canvas 绘制）来创建所有角色精灵
- 这些下载的素材是**备选/参考资源**，如需使用它们替代程序化生成，需要修改 `assets.js` 中的 `Assets` 模块，改为加载 PNG 图片而非 Canvas 绘制
- Kenney 士兵素材是**侧视角平台游戏**风格，如需俯视角（与当前游戏一致的视角），建议参考 OpenGameArt 的 `assassin_clean.png` 或 `fierce_soldier` 系列
